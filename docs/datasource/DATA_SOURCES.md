
## 二、周报数据需求（AARRR周报）

### 数据周期
- **数据周期**：上周一至上周日
- **对比周期**：上上周一至上周日
- **Acquisition/Activation特殊**：对比前两周 vs 上上两周（因转化完成周期延迟）
- **Retention特殊**：留存率对比前两周 vs 上上两周（因次周留存需要完整一周观察期）

---

### 1. Acquisition / 流量获取

#### 整体表现
- 总新访客: {number}人
- 总注册数: {number}人
- 整体转化率: {percentage}%

#### 环比上周变化
- 新访客: ↑/↓ {percentage}%
- 注册数: ↑/↓ {percentage}%
- 注册转化率: ↑/↓ {percentage}% 环比↑/↓{pp}个百分点

#### 注意项（异常渠道标注）

**每周必看渠道**（优先级最高）：
- **paid ads**: 关注投放效果，重点检测"量增质降"（访客增长但转化率下降）
  - 示例：渠道访客增长15.5%，但转化率下降1.2pp，出现量增质降，需优化投放素材/定向
- **organic search**: 关注SEO效果，检测访客量变化和核心关键词排名
  - 示例：渠道访客量下降3.9%，需关注SEO效果及核心关键词排名

**其他异常渠道**：
- mobile app search: 渠道访客变化{x}%，转化率变化{x}pp
- affiliate & kol: 渠道访客变化{x}%，转化率变化{x}pp
- social media: 渠道访客变化{x}%，转化率变化{x}pp

**判定规则**：
1. paid ads 和 organic search 每周必须生成注意项说明
2. 访客量变化 ±30% 或转化率变化 ±5pp 的渠道需标注
3. 访客增长但转化率下降的渠道（量增质降）需重点标注

### 2. Activation / 激活转化

> **注意**：Activation数据对比逻辑为"转化完成的周"，故对比前两周数据（如 2026/02/23 vs 2026/02/16）

#### 完整数据对比

| 步骤 | 当前周期 | 对比周期 | 环比变化 |
|------|---------|---------|---------|
| 新注册用户数 | 25,399 | 22,547 | ↑ 12.63% |
| 注册→进工具 | 20,651 (81.31%) | 18,110 (80.32%) | ↑ 0.99pp |
| 进工具→画户型 | 13,180 (63.82%) | 11,415 (63.03%) | ↑ 0.79pp |
| 画户型→拖模型 | 4,469 (33.91%) | 3,939 (34.51%) | ↓ 0.60pp |
| 拖模型→渲染 | 1,475 (33.01%) | 1,237 (31.40%) | ↑ 1.61pp |
| **总转化率** | **17.60%** | **17.47%** | **↑ 0.13pp** |

#### 历史趋势（近4周）

| 周期 | 新注册 | 注册→进工具 | 进工具→画户型 | 画户型→拖模型 | 拖模型→渲染 | 总转化 |
|------|--------|-----------|-------------|-------------|-----------|--------|
| 2026/02/02 | 28,047 | 76.28% | 62.80% | 33.67% | 32.59% | 16.13% |
| 2026/02/09 | 25,411 | 76.77% | 63.20% | 32.76% | 31.39% | 15.89% |
| 2026/02/16 | 22,547 | 80.32% | 63.03% | 34.51% | 31.40% | 17.47% |
| 2026/02/23 | 25,399 | 81.31% | 63.82% | 33.91% | 33.01% | 17.60% |

#### 关键发现
- 渲染转化率大幅提升：31.40% → 33.01%，↑1.61pp
- 总转化率连续提升：17.47% → 17.60%，环比↑0.13pp

**Metabase查询需求**：
- 各转化步骤的用户数
- 各步骤转化率
- 近4周历史趋势数据

### 4. Revenue / 收入分析

#### 格式要求

第一段（总体概况）：
```
当周收入 {amount} 美元，较上周 ↑/↓ {amount}美元，增长率 {+/-percentage}%。
其中，代理收入{增加/减少} {amount} 美元（增长率 {percentage}），正常收入{增加/减少} {amount} 美元（增长率 {percentage}）。
```

分项数据（bullet list）：

**账单分层（二级）：**
- 连续续约（{+/-amount} 美元）
- 新签（{+/-amount} 美元）[正向增长标注]
- 升级（{+/-amount} 美元）[正向增长标注]

**有效国家：**
- 新加坡（{+/-amount} 美元）[正向增长标注]
- 日本（{+/-amount} 美元）[正向增长标注]
- 马来西亚（{+/-amount} 美元）[负向标注]

**SKU：**
- elite_year_cyclical（{+/-amount} 美元）[正向增长标注]
- pro_month_cyclical（{+/-amount} 美元）[正向增长标注]
- pro_year_cyclical（{+/-amount} 美元）[负向标注]

#### 示例
```
当周收入 79,157 美元，较上周 ↑ 26,079美元，增长率 +49%。
其中，代理收入增加 2,600 美元（增长率 0 -> 2,600），正常收入增加 23,479 美元（增长率 +44%）。

账单分层（二级）：
- 连续续约（+14,427 美元）
- 新签（+4,329 美元）[正向增长]
- 升级（+3,382 美元）[正向增长]

有效国家：
- 新加坡（+2,128 美元）[正向增长]
- 日本（+1,954 美元）[正向增长]
- 马来西亚（+1,719 美元）[正向增长]

SKU：
- elite_year_cyclical（+7,789 美元）[正向增长]
- pro_month_cyclical（+6,433 美元）[正向增长]
- pro_year_cyclical（+5,287 美元）[正向增长]
```

**Metabase查询需求**：
- 总收入（代理收入 + 正常收入）
- 账单分层（连续续约/新签/升级/召回）
- 有效国家收入分布
- SKU收入分布

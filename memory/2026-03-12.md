## 🔑 关键知识点

**Model 层设计**: 
- 用 SQL 创建 (type: native)
- 对接数仓 DWD 层 (明细宽表)
- 上层 Question 用 MBQL (type: query) 做聚合

**MBQL 核心结构**:
```
source-table → joins → filter → breakout → aggregation → expressions → order-by
```

**条件聚合语法**:
```json
["aggregation-options",
  ["distinct",
    ["if", [[["not-null", ["field", "pay_success_date", {...}]],
              ["field", "user_id", {...}]]]]],
  {"name": "支付成功人数"}]
```
